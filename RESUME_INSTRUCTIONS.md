# Інструкція: Як продовжити завантаження після перезавантаження комп'ютера

Якщо ви вимкнули комп'ютер і хочете продовжити з 7050-го документа:

### 1. Відкрийте ПЕРШИЙ термінал і запустіть тунель
Це потрібно для зв'язку з базою даних на Hostinger:
```bash
./scripts/db-tunnel.sh
```
*(Тримайте це вікно відкритим)*

### 2. Відкрийте ДРУГИЙ термінал і запустіть сайт
Це потрібно для роботи API, який обробляє тексти:
```bash
npm run dev
```
*(Тримайте це вікно відкритим)*

### 3. Відкрийте ТРЕТІЙ термінал і запустіть скрипт завантаження
Вкажіть діапазон документів (наприклад, з 7050 по 7500):
```bash
node scripts/ingest-kcs.mjs --start=7050 --end=7500
```

**Особливості оновленого скрипта:**
- **Рандомні паузи:** Скрипт автоматично робить паузи (**3с, 4с, 6с або 7с**) між кожним документом.
- **Послідовність:** Запити йдуть суворо по одному, без паралельності.
- **Ліміт:** Скрипт зупиниться автоматично після 500 звернень за один запуск (для безпеки).

---

### Що робити, якщо пише "No texts fetched"?
Це ознака того, що Реєстр Судів тимчасово обмежив доступ (захист від ботів).
1. **Зробіть паузу** на 30-60 хвилин.
2. Запустіть скрипт знову з наступного індексу.
   *Затримки тепер вбудовані в скрипт і вибираються рандомно, тому змінювати код більше не потрібно.*
